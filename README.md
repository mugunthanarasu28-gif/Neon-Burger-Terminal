Welcome to the year 2077. **NeonBurger** is a full-stack inventory management system designed for the high-tech, low-life aesthetic. This terminal allows street-vendors to manage their digital-meat inventory with a glowing, retro-futuristic interface.

## ✨ Features

-   **📡 Real-Time CRUD**: Create, Read, and Delete menu items directly from the holographic dashboard.
-   **🔥 Spicy Scaling**: Dynamic logic that translates numerical heat levels into visual 🔥 indicators.
-   **💎 Neon UI**: A custom-built CSS grid system featuring "glassmorphism," neon glows, and cyberpunk color palettes (#f107bd & #00fbff).
-   **🛡️ Robust Backend**: Powered by a relational MySQL database with a clean PHP data-access layer.

## 🚀 Installation & Setup

To get the terminal online in your local sector:

1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/mugunthanarasu28-gif/NeonBurger.git](https://github.com/mugunthanarasu28-gif/NeonBurger.git)
    ```
2.  **Move to Web Root:**
    Move the `neon` folder to your `C:/xampp/htdocs/` directory.
3.  **Database Injection:**
    - Open `phpMyAdmin`.
    - Create a database named `neonburger_db`.
    - Import the `database.sql` file (or paste the SQL code provided in the project).
4.  **Launch:**
    Start Apache and MySQL in XAMPP and visit:
    `http://localhost/neon/index.php`
